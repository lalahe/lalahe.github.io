什么是高阶组件

  • 官网介绍说，高阶组件（HOC）是 React 中用于复用组件逻辑的一种高级技巧。HOC 自身不是 React API 的一部分，它是一种基于 React 的组合特性而形成的设计模式。具体而言，高阶组件是参数为组件，返回值为新组件的函数。
  • 简单点说，高阶组件其实就是处理react组件的函数

怎么使用高阶组件

  • 实现高阶组件的方法有两种
    1. 属性代理（这种方法是最常见的实现方式，将被处理的props和新的props一起传递给新组件）
    2. 反向继承
